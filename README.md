# 短链接项目

## 项目简介

短链接项目旨在提供一个简单、可靠的短链接生成和管理服务。用户可以将长网址转换为短链接，并跟踪短链接的点击统计信息。

## 功能特性

- **短链接生成**：将长网址转换为简短易记的短链接。
- **点击统计**：统计每个短链接的点击次数。
- **链接管理**：查看、编辑和删除已生成的短链接。

## 技术栈

- **后端**：Spring Boot
- **数据库**：MySQL/Redis
- **消息队列**：Rabbitmq
- **容器化**：Docker
- **权限框架**：Satoken

## 环境要求

- JDK 21
- Maven 3.6+
- Docker (可选)
- MySQL

## 分布式锁
采用lua+redis实现单机版redis分布式锁

## 限流
采用固定窗口（计数器算法）实现

## 前端
基于vue+antd实现，目前完成登录功能联调
